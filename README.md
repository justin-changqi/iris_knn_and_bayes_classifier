# Taipei Tech Machine Learning homework 1 source code
## System Environment
- Ubuntu16.04 LTS
- Python 3.0
- Libaraies: Pandas, Numpy, matplotlib

## Install Libaraies in Unbuntu for Python3
```
sudo pip3 install pandas
sudo pip3 install numpy
sudo pip3 install matplotlib
```
